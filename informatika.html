Informatika

Pembelajaran informatika yang meliputi HTML dan CSS bertujuan untuk membekali siswa dengan pengetahuan dan keterampilan dasar dalam bidang pengembangan web. HTML adalah bahasa markup yang digunakan untuk membuat struktur dan konten halaman web, sedangkan CSS adalah bahasa lembar gaya yang digunakan untuk mengatur tampilan visual halaman web.
Dengan mempelajari HTML dan CSS, siswa dapat membuat halaman web yang menarik dan informatif. Siswa juga dapat mempelajari bagaimana menggunakan HTML dan CSS untuk membuat berbagai elemen web, seperti teks, gambar, tabel, dan form.

"HTML adalah singkatan dari HyperText Markup Language, yaitu bahasa standar untuk membuat halaman web12. HTML mendeskripsikan struktur dan makna konten web dengan menggunakan serangkaian elemen12. Elemen HTML terdiri dari tag pembuka, tag penutup, dan konten di antaranya. Contoh elemen HTML adalah "p" untuk paragraf, "h1" untuk judul, "img" untuk gambar, dan lain-lain."
